## Running App

**Client**

just first time installation:
```
yarn install
```

run app:
```
yarn start
```

**Api**

Linux/Mac OS:
```
json-server trickygames.db.json --watch --port 3001
```
Windows:
```
npx json-server trickygames.db.json --watch --port 3001
```
